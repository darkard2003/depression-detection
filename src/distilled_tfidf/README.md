# Distilled TF-IDF Student Model

This directory contains the **Distilled TF-IDF** student model, which represents our baseline lexical distillation setup. It attempts to fit the teacher's prediction logits using sparse word features.

---

## 1. Feature Representation
* **Feature Type:** Sparse term-frequency inverse-document-frequency (TF-IDF).
* **Dimensionality:** Selected top **5,000 features** via Chi-Square (`chi2`) selection from the lexical corpus.

---

## 2. Training Target (Distillation)
Unlike models trained directly on labels (0 or 1), the Distilled TF-IDF model is trained using **soft targets** generated by the teacher model (`TRT1000/depression-detection-model`):
* **Fidelity Targets:** Fits the floating-point probability outputs of the teacher rather than discrete ground-truth labels.
* **Loss Function:** Trained using Binary Crossentropy against the soft targets to mimic the teacher's probability boundaries.

---

## 3. Scripts
* [get_teacher_labels.py](file:///Users/dark/code/project/depression/src/distilled_tfidf/get_teacher_labels.py): Extracts soft predictions from the teacher model for the training dataset.
* [train_distilled.py](file:///Users/dark/code/project/depression/src/distilled_tfidf/train_distilled.py): Runs Keras Tuner Hyperband search and fits the student MLP using the extracted soft targets.
* [compare_predictions.py](file:///Users/dark/code/project/depression/src/distilled_tfidf/compare_predictions.py): Compares the student's predictions against teacher logits to assess agreement and computes inference latency.

---

## 4. Performance Results
* **Fidelity Agreement (with Teacher):** **69.01%** (F1: **0.54020**) on degraded [bin_reddit1.csv](file:///Users/dark/code/project/depression/datasets/bin_reddit1.csv).
* **Inference Latency:** **~0.20 ms/sample** on CPU.
